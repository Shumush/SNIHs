# SNIHs
[SHAUN](https://github.com/ablanleuil/SHAUN) (implementation) In Haskell

# Build
## Cabal
```
    cabal configure
    cabal build
    cabal install
```
## ghc-pkg
```
    runhaskell Setup.hs configure --ghc
    runhaskell Setup.hs build
    runhaskell Setup.hs install
```
## Haskell Stack
```
    stack install
```
